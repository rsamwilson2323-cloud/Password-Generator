# 🔐 Password Generator

A modern and secure Password Generator web application built with HTML, CSS, and JavaScript. Generate strong, random, and customizable passwords instantly to improve online security. The application offers an easy-to-use interface, responsive design, and fast password generation without requiring any external dependencies.

## ✨ Features

* 🔐 Generate strong random passwords
* 🎯 Customizable password length
* 🔡 Uppercase letter support
* 🔠 Lowercase letter support
* 🔢 Number inclusion option
* ✨ Special character support
* 📋 One-click password copy
* 📱 Mobile-friendly interface
* ⚡ Instant password generation
* 🚀 Lightweight and dependency-free

## 🎮 How to Use

1. Open the Password Generator application.
2. Select your desired password settings.
3. Choose the password length.
4. Enable or disable character options.
5. Click the Generate button.
6. Copy the generated password with a single click.
